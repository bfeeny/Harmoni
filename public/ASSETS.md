### Sound File Requirements

For optimal performance in the web application:

- File format: MP3
- Bit rate: 128-192 kbps (balance of quality vs file size)
- Duration: 1-3 minutes (loops seamlessly)
- Normalize to -3dB peak level for consistent volume across sounds
